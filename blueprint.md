# Project Blueprint

## Overview

This project is a simple lottery number generator. It provides a user-friendly interface to generate and display random lottery numbers.

## Features

*   **Lottery Number Generation:** Generates a set of 6 unique random numbers between 1 and 45.
*   **Interactive Interface:** A button to trigger the number generation.
*   **Modern Design:** A clean and visually appealing layout.

## Current Task

### Create a lottery number recommendation website.

*   **index.html**: Add a title, description, a container for the lottery numbers, and a button to generate new numbers.
*   **style.css**: Add styles for the body, container, button, and lottery numbers.
*   **main.js**: Implement the logic to generate 6 random numbers between 1 and 45 when the button is clicked and display them.
